## Installation

### iPhone (iOS)
1. Réglages → Données cellulaires → Ajouter un forfait data
2. Scanner le QR code ci-dessous
3. Activer l'itinérance des données sur cette eSIM
4. [Vidéo tuto](https://youtu.be/NvX16WFjnP8?si=SPgYYdKKP6CMvr61)

### Android
1. Paramètres → Réseau & Internet → + à côté de Réseau mobile
2. Scanner le QR code
3. Activer l'itinérance des données sur cette eSIM
4. [Vidéo tuto](https://youtu.be/xgxalF5U5AE?si=68Y-APJYKPUBG0Tp)
